Week 1

What types of native features (on Android and iOS devices) would you be able to implement with the use of HTML/CSS/JS only? (just a hint - iOS can be the tricky one)

The camera, audio, geolocation, contacts, video, and accelerometer can be implemented via Phonegap.

For your CRUD app revision, which of these do you think you will implement?

I plan on replacing my javascript alerts with native notifications. I also would like to utilize the audio feature.

Briefly describe some past 'lessons learned' regarding workflow as it relates to research, design, development, etc.

When working on a project is essential to have a positive mindset and to try to be in a good mood. I noticed if I'm stressed or trying to meet a deadline, my quality of work tends to suffer.

Please describe any design intentions you may have for the AVF Demo app and the revised/new version of your CRUD app.

I would like to try to make a different app completely. I'm a little tired of the same app I've been polishing from the original VFW class. If I do not have time I will implement some native Phonegap features into my current app.

If not covered above, please also describe any functionality intentions you may have for both the AVF Demo app and the revised/new version of your CRUD app.

I want to make sure the structure of my HTML code to accessible via a screen-reader, while being WC3 compliant.
