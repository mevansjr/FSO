Scouting Inc


This App is designed for Scouts to evaluate Sport Players. The Scout can create and store a Player in a database. The Player can be rated on a numeric level, along with some other criteria. After a Player has been created, you can go back to the Player and edit some information or delete the Player. There are various features available, so enjoy!
